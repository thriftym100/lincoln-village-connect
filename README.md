# Lincoln Village Connect Premium

Premium front-end design for the Lincoln Village Thrifty Wash guest Wi-Fi portal.

## Required image

Place the existing hero image here:

`images/hero-background.png`

## Preview

Open `index.html` with Live Server in VS Code.

## Deploy

Commit and push through GitHub Desktop. Cloudflare Pages will deploy automatically.

## Important

The form currently demonstrates front-end validation only. Mailchimp and UniFi credentials must be handled server-side through Cloudflare Pages Functions.
