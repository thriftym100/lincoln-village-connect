# Lincoln Village Connect

Version 1 of the Lincoln Village Thrifty Wash guest Wi-Fi portal.

## Local preview

Open the folder in VS Code, right-click `index.html`, and choose **Open with Live Server**.

## Deployment

Commit and push changes through GitHub Desktop. Cloudflare Pages will redeploy automatically.

## Important

The current form is a front-end demonstration. Do not place Mailchimp API keys or UniFi credentials in `app.js`. Those integrations belong in server-side Cloudflare Pages Functions.
